# CAMINHADA-ALEATORIA-Z² 
Este repositório reúne códigos, simulações e gráficos desenvolvidos para estudar propriedades de caminhadas aleatórias simples em duas dimensões. Os experimentos incluem:
Probabilidade de retorno à origem, Convergência da estimativa de Monte Carlo, Decaimento da probabilidade de retorno, Tempo de cobertura de regiões quadradas no plano
O objetivo é compreender empiricamente o comportamento de processos estocásticos clássicos através de simulações numéricas.
